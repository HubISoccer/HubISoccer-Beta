# HubISoccer - Site d'inscription des joueurs

## Description
Ce projet est un site web destiné à gérer les inscriptions des joueurs pour le tournoi HubISoccer 2026.  
Le site permet :

- L'inscription des joueurs libres ou en club
- L'envoi de vidéos montrant leurs capacités
- La sélection des joueurs en fonction du profil recherché
- La génération d'une carte de licence pour les joueurs sélectionnés
- La diffusion de la FAQ et des informations sur les postes recherchés
- La visualisation des maillots avec choix de couleur

Le site est conçu pour être **responsive** et facile à utiliser sur mobile et ordinateur.
